Technical Artist 是做什麼的？

TA = 「懂程式的藝術家」+ 「懂美術的工程師」

⚠ 沒有 TA 的遊戲
顏色平坦、沒有光影、角色像紙片人。程式設計師做不出漂亮效果，美術師又不懂程式……
✨ 有 TA 的遊戲
發光材質、粒子特效、流暢動畫。TA 橋接了兩個世界，讓遊戲「活起來」！
✨
Shader 魔法師
用程式碼「畫」出材質：金屬的反光、水面的波紋、火焰的顏色——都是 Shader 寫的
HLSL / GLSL
🔧
工具打造者
寫程式讓美術師「一鍵完成」原本要手動做幾小時的工作，大家效率更高
Python / Maya Script
🌊
特效設計師
爆炸、火焰、魔法光球——這些視覺特效（VFX）讓遊戲充滿衝擊感
Houdini / Niagara
🦴
角色骨架師
建立角色的「骨架系統」，讓動畫師可以讓角色跑、跳、揮劍——動作自然流暢
Rigging / Skinning

5 個關卡，從零到入門

每個關卡大約 20 分鐘，按順序玩最好

🎨
1
顏色魔法
RGB 是什麼？改數字讓顏色變！
電腦的顏色是三個數字組成的（紅、綠、藍）。試著改數字，看看會發生什麼事——這就是 Shader 的開始！
🖥 瀏覽器直接玩 不需安裝任何東西
⏱ 約 15 分鐘
〜
2
動起來！
讓東西動起來——sin 和 cos
遊戲裡所有「波浪、搖晃、呼吸感」都是 sin 做的。只要一個數學函數，就能讓靜止的東西活過來！
📐 數學概念 🖥 動態預覽
⏱ 約 20 分鐘
✨
3
粒子特效
用 1000 個點做出火焰和雪花
特效不是一張圖，是「很多很多小點一起動」！你來控制速度、大小、顏色——設計你自己的特效！
🌨 雪花系統 🔥 火焰效果 🖥 即時互動
⏱ 約 20 分鐘
🔧
4
自動工具
寫一個幫你省時間的小工具
「這件事要做 100 次，好煩……」——TA 的反應是「那就寫個程式自動做！」。體驗一下 Python 自動化的魔力。
🐍 Python 入門 ⚡ 自動化概念
⏱ 約 25 分鐘
🏆
5
畢業作品
你的第一個 Shader！
把前面學的全部用上——做出一個發光、會動的材質球。這就是你的第一份 TA 作品！截圖分享給朋友看吧 🎉
🌐 Shadertoy 📸 可以截圖分享
⏱ 約 30 分鐘
