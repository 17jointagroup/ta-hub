2-2 基礎光照 — Phong 的三層魔法

Ambient + Diffuse + Specular = 真實感

+120
✓
📖故事
🎮互動
💻程式碼
1975 年，Bui Tuong Phong 將光拆成三層：

Ambient 環境光 — 場景底色，防止全黑。
Diffuse 漫反射 — 角度越正對光源越亮。
Specular 高光 — 鏡面反射亮點。
PHONG 光照 — 即時球體
✦ 光澤度 32
vec3 N=normalize(vN), L=normalize(uLight-vP), V=normalize(uEye-vP), R=reflect(-L,N);
vec3 ambient = 0.18 * objColor;
vec3 diffuse = max(dot(N,L),0.0) * objColor;
float spec = pow(max(dot(V,R),0.0), uShininess);
gl_FragColor = vec4(ambient+diffuse+spec*vec3(1.0),1.0);

方向光（Directional）：無限遠，所有光線平行，如太陽。

4-1 深度測試 — 誰在前面？

Z-buffer：解決物體重疊時的遮擋問題

+70
📖故事
🎮互動
💻程式碼
GPU 為每個像素儲存一個深度值（Z-buffer）。畫新片段時比較深度，只保留更近的，遠的直接丟棄。

關閉深度測試後，後畫的物體會覆蓋先畫的，無論遠近。
深度測試 — Z-BUFFER 可視化
glEnable(GL_DEPTH_TEST);  // 開啟
glDepthFunc(GL_LESS);   // 預設：近的優先
glClear(GL_DEPTH_BUFFER_BIT); // 每幀清除
