OpenGL 渲染管線把 3D 資料變成螢幕上的像素，主要步驟：

CPU 準備頂點 → VBO 上傳 GPU → Vertex Shader 變換位置 → Rasterize 插值 → Fragment Shader 著色 → 輸出 Framebuffer。
渲染管線 — 即時三角形
attribute vec3 aPos;
void main(){ gl_Position = vec4(aPos, 1.0); }

// Fragment Shader
precision mediump float;
void main(){ gl_FragColor = vec4(1.0,0.5,0.0,1.0); }

GLSL 是 GPU 的語言。uniform：CPU 傳給 GPU 的參數（時間、光源）。attribute：每頂點的資料。varying：VS 到 FS 的插值。
GLSL 著色器 — 即時顏色控制
uniform float uTime;
float r=sin(uTime)*0.5+0.5;
float g=sin(uTime+2.09)*0.5+0.5;
gl_FragColor=vec4(r,g,0.8,1.0);
