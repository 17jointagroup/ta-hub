4-1 深度測試 — 誰在前面？

Z-buffer：解決物體重疊時的遮擋問題

+70
📖故事
🎮互動
💻程式碼
GPU 為每個像素儲存一個深度值（Z-buffer）。畫新片段時比較深度，只保留更近的，遠的直接丟棄。

關閉深度測試後，後畫的物體會覆蓋先畫的，無論遠近。
深度測試 — Z-BUFFER 可視化
glEnable(GL_DEPTH_TEST);  // 開啟
glDepthFunc(GL_LESS);   // 預設：近的優先
glClear(GL_DEPTH_BUFFER_BIT); // 每幀清除

4-2 模板測試 — 蒙版魔法

Stencil Buffer：選擇哪些像素要被畫

+80
📖故事
🎮互動
💻程式碼
Stencil Buffer 是一個遮罩。先把物體輪廓寫入 Stencil，再用 Stencil 遮罩畫一個放大版本 —— 就得到了發光邊框。
模板測試 — 物體發光邊框
🪡 邊框厚度 14
// Pass 1: write stencil
glStencilFunc(GL_ALWAYS, 1, 0xFF);
draw(object);

// Pass 2: draw outline where stencil==0
glStencilFunc(GL_NOTEQUAL, 1, 0xFF);
draw(scaledObject);

4-7 幾何著色器 — 憑空生成三角形

Geometry Shader：爆炸特效、草地、法向量視覺化

+100
📖故事
🎮互動
💻程式碼
Geometry Shader 在 VS 和 FS 之間，可以新增或刪除圖元。

爆炸特效：把每個三角形沿法向量方向推出去。法向量視覺化：每個頂點生成一條線段。
幾何著色器模擬 — 爆炸特效
// Geometry Shader — explode
layout(triangles) in;
layout(triangle_strip, max_vertices=3) out;

vec3 n = GetNormal();
for(int i=0;i<3;i++){
gl_Position = gl_in[i].gl_Position + vec4(n*time,0);
EmitVertex();
}

4-10 進階 GLSL 技巧集

深度非線性、法向量貼圖、視差貼圖一次看

+80
📖故事
🎮互動
💻程式碼
法線貼圖（Normal Map）：用 RGB 貼圖儲存每個像素的法向量，低多邊形模型模擬高細節表面凹凸。

RGB → XYZ：R→X（左右）、G→Y（上下）、B→Z（深度）。
法線貼圖 — 假的凹凸感
// Sample normal map
vec3 N = texture(uNormalMap, vUV).rgb;
N = normalize(N * 2.0 - 1.0); // [0,1]→[-1,1]
N = normalize(TBN * N); // Tangent→World space
